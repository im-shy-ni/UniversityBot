# University Chatbot System

The University Chatbot System is a web-based application designed to provide instant responses to student queries related to university information such as admissions, courses, fees, departments, and general inquiries. The chatbot improves accessibility by delivering quick and accurate information without manual assistance.

## Features

### User Features
- Interactive Chat Interface
- Instant Responses to Queries
- Provides Information on Courses & Departments
- Admission Details & Guidelines
- Frequently Asked Questions (FAQ) Support
- User-Friendly and Responsive Design

### Chatbot Features
- Predefined Knowledge Base (JSON/Data-driven)
- Keyword-based Response System
- Quick Query Processing
- Easy to Update Responses
- Handles Multiple User Queries
